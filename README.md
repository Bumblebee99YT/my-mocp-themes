# Instructions
1. Make sure you have [**mocp**](http://moc.daper.net/ "mocp website") installed
2. Copy paste the theme files into `/usr/share/moc/themes`
3. Open mocp in terminal by typing `mocp`
4. Hold `Shift + T`
5. Scroll to the theme and then hit enter
6. Enjoy
